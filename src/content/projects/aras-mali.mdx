**Aras Mali Müşavirlik** için geliştirdiğim kurumsal web; muhasebe, vergi, bordro ve danışmanlık hizmetlerini modern bir yüzeyde toplar. Admin paneli GitHub bilmez — ofis sahibi `/studio` üzerinden e-posta/şifre ile içerik yönetir.

### Ürün yüzeyi

- Ana sayfa: hero, hizmet özeti, yazılar, SSS, CTA
- Hizmetler: dinamik slug detayları (SEO)
- Blog + vergi takvimi: Sanity collection’ları
- Hesaplayıcılar: KDV, net–brüt maaş, stopaj
- İletişim / randevu: Zod + Resend Server Action, WhatsApp

### Mimari

1. Sanity Content Lake + ISR / revalidate
2. Typed fallback içerik (CMS bağlanana kadar ayakta kalma)
3. Server-first sayfalar; animasyonlar izole client adalarında
4. metadata, sitemap, robots, JSON-LD

### Durum

Geliştirme devam ediyor. Production env ve Sanity bağları tamamlandıkça yayınlanacak.
