**Aras Mali Müşavirlik** için geliştirdiğim kurumsal web ürünü; muhasebe, vergi danışmanlığı, bordro, şirket kuruluşu ve finansal danışmanlık hizmetlerini tek bir modern yüzeyde toplar. Admin paneli GitHub gerektirmez — ofis sahibi `/studio` üzerinden e-posta/şifre ile içerik yönetir.

## Ürün yüzeyi

- **Ana sayfa:** Hero, sayaçlar, hizmet özeti, son yazılar, referanslar, SSS, CTA
- **Hizmetler:** Dinamik slug’lı detay sayfaları (SEO için kritik)
- **Blog + Vergi Takvimi:** Mevzuat ve beyanname takibi — Sanity collection’ları
- **Hesaplama araçları:** KDV, net–brüt maaş, stopaj ön hesaplayıcıları
- **İletişim / randevu:** Zod + Resend Server Action, WhatsApp butonu, harita

## Mimari kararlar

1. **Sanity Content Lake + ISR:** İçerik kaydı → webhook → Vercel revalidate; teknik bilgi olmadan yayın
2. **Fallback veri katmanı:** Sanity henüz bağlı değilken de site ayakta kalsın diye typed fallback içerik
3. **Server-first sayfalar:** Animasyonlar izole client bileşenlerde; SSG/SEO avantajı korunur
4. **SEO omurgası:** metadata şablonu, sitemap, robots, JSON-LD (Organization / FAQ)

## Durum

Geliştirme devam ediyor. Canlı domain hedefi `arasmali.com`; production env ve Sanity proje bağları tamamlandıkça yayınlanacak.
