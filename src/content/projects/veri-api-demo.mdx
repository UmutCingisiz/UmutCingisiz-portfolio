## Amaç

CRUD akışlarını öngörülebilir endpoint’ler ve tutarlı hata gövdeleri ile sunmak.

## Mimari notları

- Kaynak başına rota grupları
- Zod veya eşdeğeri ile giriş doğrulaması
- Idempotent işlemler için uygun HTTP fiilleri

## Kod özeti

```ts
export async function GET(req: Request) {
  const url = new URL(req.url);
  const id = url.searchParams.get("id");
  if (!id) return Response.json({ error: "missing id" }, { status: 400 });
  return Response.json({ ok: true });
}
```
